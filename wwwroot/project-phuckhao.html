Hệ thống hỗ trợ phúc khảo

ASP.NET Core, Entity Framework, SQL Server, HTML/CSS/JavaScript, Bootstrap, C#, Visual Studio 2022

Đồ án cơ sở: Công nghệ phần mềm
Website quản lý toàn diện quy trình phúc khảo điểm thi cho sinh viên đại học. Hệ thống hỗ trợ 4 vai trò: sinh viên (nộp đơn, theo dõi), giảng viên (chấm bài), nhân viên (quản lý, phân công), admin (quản trị hệ thống). Tích hợp email notification và quản lý file bài thi số hóa. Đây là đồ án cơ sở nhóm 3 thành viên với sự hướng dẫn của giảng viên bộ môn Công nghệ phần mềm.

Tính năng chính

  • Quản lý đa vai trò: Sinh viên, Giảng viên, Nhân viên, Admin với phân quyền rõ ràng
  • Nộp đơn phúc khảo: Chọn môn học, nhập lý do, upload tài liệu trực tuyến
  • Theo dõi trạng thái: Từ nộp đơn đến nhận kết quả cuối cùng
  • Phân công giảng viên: Tự động phân công dựa trên môn học
  • Chấm bài số hóa: Upload file bài thi, chấm lại, nhập điểm mới
  • Thông báo email: Tự động gửi email cho sinh viên và giảng viên
  • Báo cáo thống kê: Số lượng đơn, kết quả phúc khảo theo học kỳ
  • Quản lý đợt phúc khảo: Mở/đóng đợt theo học kỳ

Giao diện hệ thống

  • Dashboard Admin: Quản lý người dùng, đợt phúc khảo, thống kê
  • Giao diện sinh viên: Nộp đơn, xem điểm, theo dõi trạng thái
  • Giao diện nhân viên: Duyệt đơn, phân công, upload bài thi
  • Giao diện giảng viên: Nhận phân công, chấm bài, nhập kết quả
  • Hệ thống thông báo: Email tự động cho tất cả bên liên quan

Công nghệ sử dụng

  • ASP.NET Core: Framework web hiện đại
  • Entity Framework: ORM cho quản lý database
  • SQL Server: Hệ quản trị cơ sở dữ liệu
  • HTML/CSS/JavaScript: Frontend development
  • Bootstrap: Framework CSS responsive
  • C#: Ngôn ngữ lập trình backend
  • Visual Studio 2022: IDE phát triển

Quy trình nghiệp vụ

  • Bước 1: Sinh viên đăng nhập và nộp đơn phúc khảo
  • Bước 2: Nhân viên tiếp nhận và duyệt đơn
  • Bước 3: Upload bài thi số hóa lên hệ thống
  • Bước 4: Phân công giảng viên chấm bài
  • Bước 5: Giảng viên chấm lại và nhập kết quả
  • Bước 6: Thông báo kết quả cho sinh viên

Kết quả đạt được

  • Đơn giản hóa quy trình phúc khảo điểm thi, giảm thiểu giấy tờ
  • Tạo ra hệ thống minh bạch, rõ ràng, nâng cao sự tin tưởng
  • Giúp cán bộ quản lý dễ dàng giám sát, phân công, tổng hợp
  • Góp phần thúc đẩy quá trình chuyển đổi số trong quản lý học vụ
  • Hệ thống hoạt động ổn định với 4 vai trò người dùng
  • Tích hợp thành công email notification và quản lý file

Đối tượng sử dụng

  • Sinh viên: Nộp đơn, theo dõi trạng thái, nhận kết quả
  • Giảng viên: Nhận phân công, chấm bài, nhập kết quả

  • Thông báo tự động: Email cho tất cả bên liên quan
  • Quản lý file: Upload và quản lý bài thi số hóa
  • Báo cáo chi tiết: Thống kê theo học kỳ, môn học
  • Responsive: Hoạt động tốt trên mọi thiết bị
